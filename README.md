# LEGO Collection
This is a website that showcases a few LEGO sets using Express.js. The website features:
- Viewing all LEGO sets
- View specific set
- Register an account/Login

When logged in:
- Add/Edit/Delete Sets

## View Website
- Visit the site [https://lego-collection-c5al.onrender.com](https://lego-collection-c5al.onrender.com)
  - Note: The website may take a minute to load


## Frontend
- Dynamic webpage rendering with EJS
- Tailwind and DaisyUI interface

## Backend
- MongoDB and PostgreSQL storing user and set data
- client-sessions middleware and basic bcrypt hashing for passwords
